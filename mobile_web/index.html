怎么理解这些参数

网页里中文可以正常显示，所以这里把核心概念讲清楚

HSV 和三原色的关系

RGB 是红、绿、蓝三原色混合；HSV 更适合做颜色分割。 其中 H 看“是什么颜色”，S 看“颜色有多纯”，V 看“亮不亮”。

R 红 G 绿 B 蓝

色相 H 的粗略参考

0 红，30 黄，60 绿，90 青，120 蓝，150 紫。红色经常会跨过 0，所以会出现下限比上限大的情况，这是正常的。

去噪与补洞

“去噪强度”会清掉散碎小点，“补洞强度”会把目标内部的小黑洞补起来。 如果轮廓破碎，就适当加补洞；如果满屏小杂点，就适当加去噪。

手机使用提醒

手机浏览器要调用摄像头，通常需要 HTTPS 或 `localhost`。 所以后续最好把这个网页部署到 GitHub Pages、Cloudflare Pages 或你自己的 HTTPS 服务上。
